📖 Overview

TourGo is a responsive tourism web application that provides visitors with an interactive way to explore Rajamahendravaram (Rajahmundry), Andhra Pradesh. The platform combines beautiful UI, destination discovery, trip planning, live navigation, and booking management into a single seamless experience.

The latest version focuses on delivering a premium mobile-first experience inspired by modern travel applications while keeping performance lightweight using HTML, CSS, and JavaScript.
